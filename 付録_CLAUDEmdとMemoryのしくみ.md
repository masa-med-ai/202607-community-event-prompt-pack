# 付録 — CLAUDE.md と Memory のしくみ（初心者向け）

> このワークショップで Claude が「毎回ルール通りに」動いていたのはなぜか。その種明かしをする付録である。Level 2（Persona設定）や Level 10（Code体験）とあわせて読むと理解が深まる。

---

## Claude の「記憶」は1種類ではない

Claude には性質の異なる複数の記憶のしくみがあり、混同するとつまずきやすい。まず全体像を表で押さえる。

| しくみ | 誰が書くか | いつ読まれるか | どこまで残るか | 向いている内容 |
|---|---|---|---|---|
| ① 会話（チャット）の記憶 | 対話の中で自然に | その会話の間じゅう | **その会話限り**（新しいチャットでは消える） | いま進めている作業の文脈 |
| ② CLAUDE.md | **自分**（またはClaudeに書かせる） | フォルダで作業を始めるたび**自動で** | ファイルがある限りずっと | いつも守ってほしいルール・好み |
| ③ Persona.md などの参照ファイル | 自分 | CLAUDE.md から「参照せよ」と指定したとき | ファイルがある限りずっと | 職業・文体などまとまった設定 |
| ④ Memory（自動記憶） | **Claude が自動で** | 新しいセッションの開始時 | セッションをまたいで蓄積 | 会話から学んだあなたの事情・経緯 |

---

## ① 会話の記憶 — 「そのチャット限り」

同じチャットの中なら、Claude は前のやり取りを覚えている。しかし**新しいチャットを開くとゼロに戻る**。「前に言ったのに忘れている」と感じたら、たいていこれが原因である。毎回伝えるのが面倒なルールは、②以降に書き出すのが解決策になる。

## ② CLAUDE.md — フォルダに置く「指示書」

- 作業フォルダに `CLAUDE.md` という名前のファイルを置くと、Claude はそのフォルダで作業を始めるたびに**自動で読み込み**、書かれたルールに従う。
- このワークショップで Claude が「1 Step ずつ進めて確認する」進行を守っていたのも、フォルダ内の `CLAUDE.md` に進行ルールが書いてあるからである。開いて眺めてみると、しくみがそのまま見える。
- 書く内容の例: 「回答は日本語で」「医学用語の説明は不要」「箇条書きは最小限に」「引用には第一著者・雑誌名・年を付す」など。
- **試し方（1分でできる）**: `CLAUDE.md` に好きなルールを1行足して新しいチャットで話しかけると、そのルールが効いていることを確認できる。

> 補足: CLAUDE.md には階層がある。プロジェクト（フォルダ）ごとの CLAUDE.md のほかに、全プロジェクト共通で読まれる個人用の CLAUDE.md も設定できる。フォルダごとの指示のほうがより具体的な場面で効く。

## ③ Persona.md — CLAUDE.md から参照させる設定ファイル

CLAUDE.md に何でも書くと長くなりがちなので、まとまった設定（職業・文体・好みなど）は `Persona.md` のような別ファイルに分け、CLAUDE.md に「Persona.md を参照すること」と1行書いて読み込ませる方法がある。Level 2 で作ったのがこれである。見直し・編集がしやすいのが利点。

## ④ Memory（自動記憶）— Claude が自分で書き足すメモ

- Claude には、会話の中で分かった重要なこと（あなたの状況、進行中の作業、好みなど）を**自動でメモリファイルに記録**し、次回以降のセッションで参照するしくみがある（Claude Code では `MEMORY.md` などの形で保存される。利用できる機能はアプリやプランにより異なる）。
- **CLAUDE.md との違い**はここが要点である:
  - CLAUDE.md ＝ **自分が書く**明示的なルール。書いた通りに毎回効く。
  - Memory ＝ **Claude が自動で書く**メモ。便利だが、何が記録されたかは自分で確認・編集するのが安心。
- 記録された内容は開いて読める・直せる・消せる。**患者情報などの機微情報が紛れ込んでいないか**を時々確認する習慣をつけるとよい。

---

## 使い分けの早見

- 毎回必ず守ってほしいルール → **CLAUDE.md** に自分で書く
- 職業・文体などまとまった自己紹介 → **Persona.md** に分けて CLAUDE.md から参照
- 作業の経緯を勝手に覚えておいてほしい → **Memory** に任せる（ときどき中身を点検）
- いま限りの指示 → チャットでそのまま伝えるだけでよい

## よくあるつまずき

- **「CLAUDE.md に書いたのに効かない」** → CLAUDE.md はセッション開始時に読まれる。書き換えたら**新しいチャット**で試す。
- **「別のフォルダで効かない」** → フォルダの CLAUDE.md はそのフォルダ専用。共通で効かせたいなら個人用（グローバル）設定に書く。
- **機微情報を書かない** → CLAUDE.md も Memory もファイルとして残る。患者情報・パスワード等は書き込まない。
